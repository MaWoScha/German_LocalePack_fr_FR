Französisches Sprachpaket für Magento Commerce

(Magento Community Modules - French (France) Language Pack)

(Paquete de idioma francés para Magento Commerce)

Important note on Magento Version 1.9

Magento Inc. has pre-installed with the 1.9 private, semi-automatically translated, language pack. Who wants to use the LocalePack_de_DE must first delete the app/locale/fr_FR folder so that this language pack can be installed.

Wichtiger Hinweis zur Magento Version 1.9

Magento Inc. hat mit der 1.9 ein eigenes, teilautomatisch übersetztes, Sprachpaket vorinstalliert. Wer das LocalePack_de_DE benutzen möchte, muss vorher den Ordner app/locale/fr_FR löschen, damit dieses Sprachpaket installiert werden kann.

Nota importante acerca Magento Versión 1.9

Magento Inc. tiene con la versión 1.9 su propio, semi-automáticamente traducido, paquete de idiomas como preinstalado. El que quiere usar el LocalePack_de_DE, primero debe eliminar la carpeta app/locale/fr_FR para que el paquete de idioma se puede instalar.
