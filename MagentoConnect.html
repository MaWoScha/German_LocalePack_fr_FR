Französisches Sprachpaket für Magento Commerce

(Magento Community Modules - French (France) Language Pack)

(Paquete de idioma francés para Magento Commerce)

Important note on Magento Version 1.9

Magento Inc. has pre-installed with version 1.9 a private, semi-automatically translated, language pack. Who wants to use the LocalePack_fr_FR must first delete the folder app/locale/fr_FR, so this language pack can be installed.

Note importante sur Magento Version 1.9

Magento Inc. a pré-installé avec la version 1.9 d'un privé, traduit semi-automatique, pack de langue. Qui veut utiliser le LocalePack_fr_FR doit d'abord supprimer le dossier app/locale/fr_FR, donc ce pack de langue peut être installé.

Wichtiger Hinweis zur Magento Version 1.9

Magento Inc. hat mit der Version 1.9 ein eigenes, teilautomatisch übersetztes, Sprachpaket vorinstalliert. Wer das LocalePack_fr_FR benutzen möchte, muss vorher den Ordner app/locale/fr_FR löschen, damit dieses Sprachpaket installiert werden kann.

Nota importante acerca Magento Versión 1.9

Magento Inc. tiene con la versión 1.9 su propio, semi-automáticamente traducido, paquete de idiomas como preinstalado. El que quiere usar el LocalePack_fr_FR, primero debe eliminar la carpeta app/locale/fr_FR para que el paquete de idioma se puede instalar.
